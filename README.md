# ☀️ Interactive Solar System Simulation

An interactive Solar System simulation developed using **Processing (Java)**. The application visualizes the eight planets orbiting the Sun with animated movements, realistic orbital paths, and interactive controls that allow users to explore the Solar System in real time.

---

## ✨ Features

- Animated planetary orbits
- Realistic orbital speed simulation
- Interactive zoom and camera controls
- Planet information panel
- Hover effects
- Pause and resume simulation
- Adjustable simulation speed
- Asteroid belt visualization
- Star field background
- Planetary rings and moons
- Keyboard and mouse controls

---

## 🛠 Tech Stack

- Java
- Processing
- Computer Graphics
- Animation

---

## 🎮 Controls

| Action | Control |
|--------|---------|
| Pause / Resume | Space |
| Increase Speed | ↑ |
| Decrease Speed | ↓ |
| Zoom In | + |
| Zoom Out | - |
| Reset View | R |
| Hover Planet | Mouse |
| Focus Planet | Mouse Click |
| Mouse Wheel | Zoom |
